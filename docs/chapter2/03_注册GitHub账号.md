# 注册 GitHub 账号

## 这一节你会学到什么

1. 注册一个 GitHub 账号
2. 了解基本的账号安全设置

GitHub 账号是后续将博客发布到网上的必要条件，也是你参与开源社区的"通行证"。

---

## 1. 注册账号

打开浏览器，访问 GitHub 官网（https://github.com/）。

在首页找到 **Sign up** 按钮，点击后按照提示依次填写：

- **用户名（Username）**：建议使用英文，简洁好记，后续博客地址会用到它（格式为 `用户名.github.io`）
- **邮箱地址（Email）**：用于接收验证邮件和通知
- **密码（Password）**：建议使用字母 + 数字 + 符号的组合

填写完成后，通过人机验证，点击 **Create account** 完成注册，然后前往邮箱验证邮件地址。

---

## 2. 账号安全建议

注册完成后，建议完成以下设置：

### 2.1 开启两步验证（2FA）

两步验证能有效防止账号被盗。

进入路径：**头像 → Settings → Password and authentication → Two-factor authentication**

推荐使用手机 Authenticator App（如 Google Authenticator 或 Microsoft Authenticator）进行绑定。

### 2.2 完善个人资料

进入路径：**头像 → Settings → Public profile**

建议填写：
- **Name**：显示名称（可以是中文）
- **Bio**：一句话介绍自己
- **头像**：上传一张头像，让你的博客作者信息更完整

---

## 3. 了解仓库命名规则

GitHub Pages 有一个特殊规则：

> 如果你创建一个名为 `用户名.github.io` 的仓库，GitHub 会自动将它作为你的个人主页发布。

例如，你的用户名是 `zhangsan`，那么仓库名应该是：

```
zhangsan.github.io
```

发布后，访问地址就是：

```
https://zhangsan.github.io
```

这个仓库将在第3章中创建，这里只需要记住这个命名规则。
