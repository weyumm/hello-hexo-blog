Web Lego Lab

把网页像乐高一样拼起来

这个练习页让你一边点按钮，一边观察页面变化。每次改动，都是在感受 HTML、CSS、JavaScript 分别能做什么。

新手模式
网页的三块积木

结构决定“页面里有什么”，样式决定“它看起来怎么样”，交互决定“点了以后会发生什么”。

HTML

像搭骨架，负责标题、段落、图片和按钮这些元素。

CSS

像换皮肤，负责颜色、字体、间距和布局。

JavaScript

像装机关，负责点击、切换、显示和隐藏。

你的任务

试着把这个页面变成你想要的风格，再按重置观察差异。

提示：当你看到颜色变化，通常是在体验 CSS；当你看到文字被改掉，通常是在操作 HTML 元素内容；当你点击按钮后页面立刻响应，那就是 JavaScript 在工作。

操作面板

每个按钮都对应一个小实验。点一下，看看右侧页面哪里发生了变化，再猜猜这次主要动到了 HTML、CSS 还是 JavaScript。

观察提示

  • 改“文字内容”时，重点看元素里的文案发生了什么变化。
  • 改“主题颜色”时，重点看背景、按钮、标签是不是一起变了。
  • 改“样式类”时，重点看字号、边框、阴影、背景是不是被批量换掉。
